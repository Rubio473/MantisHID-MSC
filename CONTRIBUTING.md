# Contributing

1. Create a branch from the current stable source.
2. Keep target-specific hardware values under `targets/<target>/`.
3. Preserve the fixed public USB identity `MantisUSB`.
4. Do not claim complete Hak5 DuckyScript 3 compatibility.
5. Add or update mixed regression payloads when changing the parser.
6. Test USB enumeration, eject behavior, layouts, mouse, media keys, and MSC before opening a pull request.
7. Confirm the license and provenance of any third-party code or data before adding it.
