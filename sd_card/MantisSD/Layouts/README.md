# Keyboard layout data

The `.kl` files in this directory are third-party keyboard-layout data imported from the user-supplied `sdcard.zip` associated with the ElicoftZ Flipper Zero port for M5Stack Cardputer.

They are not relicensed under the repository's MIT license. The official Flipper Zero firmware project is GPL-3.0 licensed, so a GPL-3.0 license copy is included here as a conservative redistribution notice.

Before publishing a public release, confirm the exact provenance and license notice for the specific layout archive.
