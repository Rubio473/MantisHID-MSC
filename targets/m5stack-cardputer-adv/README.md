# M5Stack Cardputer ADV target

This is the only officially tested target in MANTIS V1.0.

Open the repository root in PlatformIO and use the default environment:

```text
m5stack-cardputer-adv
```

No source or configuration files need to be edited before building.

The original M5Stack Cardputer and Cardputer v1.1 are not declared officially supported until physical USB, microSD, keyboard, display, LED, and battery tests are completed.
