# Hardware support

## Officially tested

- M5Stack Cardputer ADV

## Expected but not verified

- M5Stack Cardputer
- M5Stack Cardputer v1.1

The M5Cardputer library supports Cardputer and Cardputer ADV, but V1.0 does not declare the original hardware officially compatible until physical tests cover display, keyboard, LED, battery, microSD, Keyboard HID, Mouse HID, Consumer HID, MSC, eject, and reconnect behavior.
