# Release assets

A single `MANTIS V1.0` GitHub release should contain clearly separated assets:

```text
MantisHID_MSC_V1.0_Cardputer-ADV_IMG.bin
MantisHID_MSC_V1.0_Cardputer-ADV_IMG.zip
MantisHID_MSC_V1.0_Cardputer-ADV_RAW-SD.bin
MantisHID_MSC_V1.0_Cardputer-ADV_RAW-SD.zip
MANTISUSB_500MB.img.zip
MantisSD.zip
SHA256SUMS.txt
```

Mark IMG as recommended. Mark RAW SD as full-capacity with slower detection and stricter safe-removal requirements.
