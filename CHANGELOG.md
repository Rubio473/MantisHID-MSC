# Changelog

## MANTIS V1.0

- Added Keyboard HID, Mouse HID, Consumer Control HID, and MSC.
- Added fixed `MantisUSB` USB identity.
- Added classic DuckyScript 1.x and supported Flipper-style extensions.
- Added Mantis DuckyScript 3 Core.
- Added built-in US, ES, and LATAM layouts plus external `.kl` layouts.
- Added safe eject and deferred payload/layout synchronization.
- Added IMG storage with runtime image-size detection.
- Declared 500 MiB as the officially tested and recommended image size.
- Added a PlatformIO target structure with Cardputer ADV as the default environment.
- Released original project source code and documentation under the MIT License.
